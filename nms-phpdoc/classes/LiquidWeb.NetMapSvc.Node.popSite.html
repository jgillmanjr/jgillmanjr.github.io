$nodeProperties

$nodeProperties : array

Type

array — Contains the properties that a node can have, the type to be used, and the optionality of the property The primary key of $nodeProperties is the name of the property This primary key will contain an array that indicates the type ['type'], optionality ['required'], and the data itself ['data'] An example of this is the 'name' property: $this->nodeProperties['name']['type'] = 'string'; $this->nodeProperties['name']['required'] = TRUE; $this->nodeProperties['name']['data'] = $data;

$validRelations

$validRelations : array

Type

array — This array specifies the valid "outbound" relation(s) the node type will support and the valid type(s) as targets of a particular relationship Structure of the array: $this->validRelations['relation_type1'] = array('allowed_class1', 'allowed_class2'); $this->validRelations['relation_type2'] = array('allowed_class3'); Example for `virtIface` class: $this->validRelations['connects'] = array('virtIface'); $this->validRelations['from'] = array('physIface', 'router', 'server', 'switch', 'firewall');
